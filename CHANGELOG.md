# Changelog

## unreleased
### Added
- Initial release with MVP functionality.
- Project management: create, load, save, and reopen recent projects (`.pamproj` format).
- Campaign management panel to organise monitoring deployments, including location selection via an interactive map picker.
- SD import functionality at camaign level to load audio files from SD cards into local directories.
- Species list support: load custom species list files when creating campaigns, with optional localized species names (multiple languages).
- Audio analysis powered by BirdNET-2.4 and Perch-2.0 with configurable minimum confidence.
- Examine panel with data table displaying detections, supporting filters, column visibility toggles, auto-sizing, and exporting visible columns and rows.
- Integrated audio player with spectrogram for quick playback of detected snippets.
